# Special Features

Unlock codes for optional modules. These features are included in the app but stay hidden until enabled.

| Code | Feature | Description |
| --- | --- | --- |
| 4927 | SQL Assist | Advanced workspace for SQL tables, data sources, and saved queries. |
| 7314 | JIRA Integration | Sync and review Jira issues inside the assistant. |
| 8642 | Email Ingest | Ingest Outlook emails into searchable shards with summaries. |
| 5398 | Issue Calendar | Read-only calendar view for imported issue schedules with shared notes. |
| 4826 | Production Log | For tracking production counts, updating the client's log spreadsheet, and keeping consistent formatting of the spreadsheet. |
| 6074 | SQL Builder | AI tailored assistant to assist and build SQL queries. |
